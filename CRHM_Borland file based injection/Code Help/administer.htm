Administer -

class __declspec(dllexport) Administer {

public:

TStringList *DLLModuleList;            // modules defined in DLL

TStringList *DLLModelList;              // models defined in DLL

TStringList *DLLModelModuleList; // modules used in DLL models in CommaText

String Version;

String DLLName;

String HelpFile;

Administer(String Version, String _HelpFile);

~Administer();

void AddModule(ClassModule *Module);

void AddModel(String ModelName, String ModelModules);

void LoadCRHM(String DllName);

};

Member Functions.

Constructor.

    Creates three TStringLists to hold all the modules defined,  to hold defined models and lastly the modules used by each model.

Destructor.

    Deletes instances of all modules defined.  Deletes the three TStringLists.

void AddModule(ClassModule *Module);

    Called by function MoveModulesToGlobal() to add user module name and address to the DLLModuleList.

void AddModel(String ModelName, String ModelModules);

    Called by function MoveModulesToGlobal() to add user model name to the DLLModelList and a String holding the required user modules to DLLModelModuleList.

void LoadCRHM(String DllName);
