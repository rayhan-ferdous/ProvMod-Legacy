MoveModulesToGlobal - Loads CRHM user modules.

    This function is called by CRHMmain either loaded statically from the CRHM modules defined as the default module library or as an imported function from a dynamically linked DLL.

Static version MoveModulesToGlobal().

    MoveModulesToGlobal() is defined in "OurModules.h" and other module libraries and is specified using a simple or conditional #include.

#ifdef QUINTON

#include "quinton.h"

#include "quinton.cpp"

#else

#include "OurModules.h"

#include "OurModules.cpp"

#endif

    In __fastcall TMain::FormCreate(TObject *Sender) there is a call to MoveModulesToGlobal().  The default is the user default name.

Imported version of MoveModulesToGlobal

    In __fastcall TMain::OpenDLLFile(String FileName) the call LoadUserModules = (LoadModuleType *) GetProcAddress(DLL, "_MoveModulesToGlobal") imports the function to loaded from the dynamically linked DLL.
